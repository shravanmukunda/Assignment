# Agent Assignment System (MERN)

## Prerequisites
Node ≥ 18, MongoDB ≥ 6.0

## Setup
1. git clone …
2. cd backend && cp .env.example .env   # add your Mongo URI + JWT secret
3. npm install && npm run dev
4. open new terminal
   cd frontend && npm install
   cp .env.example .env
   npm start
5. Visit http://localhost:3000
